# Stress Test

## Installation on server

* `chmod -R 777 apps/`
* `docker compose up -d`

## Run on local computer

* `composer install`
* `./vendor/bin/pest stress http://<server_ip>:14004`

* 14001 - Laravel
* 14002 - Symfony
* 14003 - Express
* 14004 - Nest
